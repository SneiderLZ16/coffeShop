¿Como empezo esto?

Hace un par de años empezo mi amor y curiosidad por el mundo de la programación. Esta misma me apasiona porque cada día trae nuevos desafíos que me motivan a aprender y crecer profesionalmente.

Además de la tecnología, me gustan mucho los deportes. Creo que mantenerme activo y practicar deportes ayuda a tener una mente clara y estar siempre enfocado en mis objetivos.

Me encanta combinar mi pasión por la programación con un estilo de vida saludable y dinámico.

Inicie aprendiendo python, haciendo ejercicios basicos y avanzando a proyectos mas complejos. Después de un tiempo, decidí aprender a html, css y js para poder crear sitios web y aplicaciones web.

Lenguajes que utilizo

HTML

HTML (HyperText Markup Language) estructura el contenido de la web mediante etiquetas.

CSS

CSS permite aplicar estilos visuales como colores, bordes, animaciones y layouts.

JavaScript

JavaScript agrega interactividad y lógica a las páginas web, como botones dinámicos y validaciones.

Python

Lenguaje de programación ampliamente utilizado en las aplicaciones web, el desarrollo de software, la ciencia de datos y el machine learning
